    HOW TO PLAY

    Pick a set (a slice of the periodic table), a mode and a difficulty, then race — solo or against up to 5 players. Everyone gets the same questions.

    Difficulty scales the challenge to fit each mode: the number of questions (Pinpoint, Namedrop, Build), the number of name options (Line-up), or how many elements you sort at once (Mass). ALL uses the whole set. Sweep always covers the whole set.

    PINPOINT — an element's name is shown; tap its cell on the table, then confirm.

    LINE-UP — an element tile is shown; pick its name from the list.

    NAMEDROP — an element tile is shown; type its name (spelling is forgiving).

    MASS — drag the elements into atomic-mass order, lightest at the top, then confirm to reveal.

    SWEEP — type every element in the set from memory — names or symbols both count. GIVE UP to stop the clock.

    BUILD — the set's cells are blank; each round names an element — tap where it lives.

    In multiplayer, when everyone's done, tap a player's card at the top to see their answers.
